# Padel Rankings App

A web application for padel players to record matches and track ELO rankings.

## Getting Started

### Environment Setup

1. Copy the environment variables template:

```bash
cp env.example .env.local
```

2. Edit `.env.local` and add your Supabase credentials:

```
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_supabase_service_role_key  # Optional but recommended
```

### Running the Development Server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

## Features

- Record padel matches between teams of 2 players
- Track player ELO ratings
- View match history
- Leaderboard with player rankings

This project uses [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.
